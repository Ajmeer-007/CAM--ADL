CM-ADAL (Attention-Guided Adversarial Domain Adaptation for Lung Disease Classification) is a deep learning framework designed to tackle the challenge of cross-modality medical image classification. The architecture integrates a ResNet18 backbone with Convolutional Block Attention Module (CBAM) for feature refinement, paired with a Domain-Adversarial Neural Network (DANN) and Gradient Reversal Layer (GRL) to align feature distributions across CT scans and Chest X-rays. The model performs four-class pulmonary disease classification while remaining robust to domain shift — enabling knowledge transfer between imaging modalities without requiring paired data.
